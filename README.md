# LifeSaver AI 🤖

AI Productivity Assistant that helps users manage tasks, deadlines and daily planning.

## Features 🚀

✅ Create AI based daily plans  
✅ Add and manage tasks  
✅ Track deadlines  
✅ Complete tasks with Done button  
✅ Simple and user friendly interface  

## Technology Used 💻

HTML  
CSS  
JavaScript  

## How it Works ⚙️

1. User enters task and deadline
2. AI generates a simple daily plan
3. User manages tasks and completes them
